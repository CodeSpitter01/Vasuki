# TermOSINT v1.0

Free terminal OSINT tool with deep recursive dorking and exact username matching.

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python vasuki.py -cat st -subcat pro col soc -n "Name Surname" -u "username" -k "Keyword" --deep --max-depth 2
```

## Features
- Scan 50+ platforms (social, professional, tech, gaming, etc.)
- Exact username matches
- Deep web dorking (DuckDuckGo)
- Categorized output (heading‑only display)
- JSON report export
